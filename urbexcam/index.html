ABOUT URBEXCAM

UrbexCAM is a horror exploring game that you can discover a lot of mystic abondoned places and you don't know what's hiding beyond the shadows.

But there is more, there are some missions for these maps to unlock more maps in FreePlay and you can also get more experiences with it.

This is a CO-OP and Singleplayer game. It will be available on Steam soon in summer, but only available for game testers. Also there are some funny things in the game and lot of fun things.

The game is in underdevelopment.
